## Overview

QUADS is the official high-fidelity campus marketplace designed specifically for University of Mines and Technology (UMaT) students. It enables secure user registration, verified listings, local category search, in-app messaging, and a trusted buyer-seller escrow system to ensure campus transactions are completely safe.

---

## Key Features

- **Secure buyer/seller escrow transaction security**
- **Real-time chat and in-app negotiating messaging system**
- **Item photo uploading, categorization, and details editing**
- **UMaT campus-specific verification flow**

## Getting Started

### Prerequisites

Ensure you have the required runtimes installed:
- Git
- Appropriate package manager (npm/yarn or pip depending on stack)

### Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/iamroidev/quads.git
   cd quads
   ```

2. **Install Dependencies**
   *For Node/Next.js/React Native projects:*
   ```bash
   npm install
   ```
   *For Python projects:*
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Environment**
   Duplicate the env file (if present) and populate variables:
   ```bash
   cp .env.example .env
   ```

4. **Launch Application**
   *For React/Next.js:*
   ```bash
   npm run dev
   ```
   *For Expo/React Native:*
   ```bash
   npx expo start
   ```
   *For Python/Flask:*
   ```bash
   python app.py
   ```
