# QUADS Web Client 🎨

The official QUADS web application, built with a unique **Bulletin Board Design System**.

## 🚀 Live Site
[https://quadsmarket.tech](https://quadsmarket.tech)

## 🎨 Design System
- **Framework**: React + Vite
- **Styling**: Tailwind CSS (Custom Bulletin Configuration)
- **Key Features**: Polaroid product cards, rigid neobrutalist borders, and technical mono-typography.

## 📦 Deployment
The frontend is hosted on **Vercel** with a **Cloudflare** security layer.

### Environment Variables (Required in Vercel)
- `VITE_API_URL`: `https://api.quadsmarket.tech/api`
- `VITE_SOCKET_URL`: `https://api.quadsmarket.tech`
- `VITE_SUPABASE_URL`: *(Your Project URL)*
- `VITE_SUPABASE_KEY`: *(Your Anon Key)*

## 🛠️ Local Development
1. Install dependencies: `npm install`.
2. Start the dev server: `npm run dev`.
3. Build for production: `npm run build`.

## 📂 Features
- **Escrow-as-a-Service**: Secure transaction handling.
- **Student Verification**: Institutional gates for high-trust listings.
- **Price Insights**: Live market data visualization.
- **Social Marketplace**: Sold feeds and top seller leaderboards.
