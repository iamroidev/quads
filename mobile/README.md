# QUADS Mobile App 📱

The mobile extension of the QUADS platform, bringing the campus marketplace to your pocket.

## 🚀 Stack
- **Framework**: Expo (React Native)
- **Language**: TypeScript
- **Auth**: Supabase + Clerk (Future)
- **State**: React Query + Context

## 🎨 Visual Identity
The mobile app mirrors the **Bulletin Board** aesthetic of the web platform, optimized for a touch-first experience with:
- Bold, high-contrast UI components.
- Secure, biometric-ready verification flows.
- Real-time push notifications.

## 🛠️ Local Development
1. Install dependencies: `npm install`.
2. Install Expo CLI: `npm install -g expo-cli`.
3. Start the project: `npx expo start`.

## 📦 Deployment
The app is managed via **EAS (Expo Application Services)**.

### Build Commands
- **Android**: `eas build -p android`
- **iOS**: `eas build -p ios`
