# QUADS — Project Handoff (May 2026)

## 📋 System Overview
**QUADS** is a high-fidelity peer-to-peer campus commerce platform specifically tailored for institutional ecosystems (starting with UMaT).

| Component | Technology Stack | Hosting/Infrastructure |
| :--- | :--- | :--- |
| **Server** | Node.js, Express, MongoDB, Socket.io | AWS EC2 (PM2, SSL via Certbot) |
| **Web** | React, Vite, Tailwind CSS (PWA) | Vercel (Production) |
| **Mobile** | React Native, Expo | Expo Application Services (EAS) |
| **Auth** | Supabase (Exchange for custom JWT) | Supabase Cloud + Firebase (Phone) |
